# 免责声明：本仓库主打一个口述缝合

这里的插件，大多来自用户打字口述需求，然后丢给 AI 现场理解、现场拼接、现场缝合。

需求可能是清楚的，也可能是“你应该懂吧”；实现可能是优雅的，也可能是能跑就先跑；至于最后跑不跑得了，那就看环境、版本、依赖、运气，以及你有没有认真看日志了。

本仓库不承诺稳定，不承诺兼容，不承诺适合生产环境。插件能跑是缘分，跑稳是造化，跑炸了请先看配置、日志、路径、权限、版本和你自己到底填了什么。

用户只负责提出想法，不负责兜底运行结果。你要是拿去用，默认你已经接受：这些东西本质上是实验性插件，靠实机挨打进化，靠问题反馈续命。

所以，安装前请备份，升级前请留退路，重要环境别上来就梭哈。真是的，别等出事了才露出“怎么会这样”的表情。

能用就偷着乐，不能用就随缘修。
